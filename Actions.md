1. Add item to cart
    - Inputs: 
      - Product ID
      - Quantity
      - Customer ID
      - Product Customizations
    - Responses:
      - Cart ID
      - Item in Cart ID
      - List of other Item IDs in the Cart
    - Expected Result:
      - Item is added to cart with specified customizations
2. Remove item from cart
    - Inputs:
      -  Item in Cart ID
      -  Cart ID
    - Responses:
      -  Success/Error
    - Expected Result:
      - Item is removed from cart
